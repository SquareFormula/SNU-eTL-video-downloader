# SNU-eTL video downloader

## How to use?
Input URL

ex) "http://etlstream.snu.ac.kr:1935/ctlocw/_definst_/mp4:blah-blah-code/yyyy/mm/dd/blah-blah-code-file-name.mp4/"


## How to get URL?
Use DevTools and then find(Ctrl+F) "sources:"
